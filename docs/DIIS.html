FUNCTION:

  • DIIS.DIIS(F,D,S,Efock,Edens,basis,numbF)
  • return Fprime, Efock, Edens, Emax

Input:

  • F, Fock matrix
  • D, Density matrix
  • S, overlap matrix
  • Efock, saved fock matrices
  • Edens, saved density matrices
  • basis, basis set
  • numbF, number of matrixes to save

Output:

  • Fprime, F’ guess
  • Efock, saved fock matrices
  • Edens, saved density matrices
  • Emax, Maximum error in error vector
